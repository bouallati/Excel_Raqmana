# 🚀 Excel_Raqmana v1.0

**إضافة احترافية لبرنامج Microsoft Excel لتسهيل مهام الأساتذة في معالجة ملف النقاط.**

---

## 📖 حول البرنامج
تم تطوير **Excel_Raqmana** ليكون المساعد الذكي للأستاذ في منصة الرقمنة. يقوم البرنامج بتحليل علامات التلاميذ وإدراج التقديرات والإرشادات التربوية آلياً وبضغطة زر واحدة، مما يوفر ساعات من العمل اليدوي ويقلل من الأخطاء البشرية.

## ✨ المميزات الرئيسية
* **معالجة آلية:** إدراج التقديرات (ممتاز، قريب من الحسن،...) والإرشادات لكل تلميذ بناءً على معدله.
* **فحص الأخطاء:** كشف تلقائي عن العلامات غير المنطقية (أقل من 0 أو أكبر من 20).
* **دعم الفواصل:** التعامل الذكي مع الفواصل العشرية (نقطة أو فاصلة) حسب إعدادات جهازك.
* **واجهة عربية:** تبويب مخصص داخل شريط Excel (Ribbon) سهل الاستخدام.
* **مجاني بالكامل:** عمل تطوعي متاح لجميع الأساتذة لوجه الله تعالى.

## 🛠️ كيفية التثبيت
1. اذهب إلى صفحة [Releases](رابط_صفحة_التحميل_هنا) وقم بتحميل آخر نسخة مضغوطة.
2. قم بفك الضغط عن الملف وتشغيل `setup.exe`.
3. وافق على رسائل التثبيت (قد تظهر رسالة تحذير لأن الناشر مستقل، اضغط More Info ثم Run Anyway).
4. افتح برنامج Excel وستجد تبويب **"الرقمنة"** قد ظهر في الأعلى.

## ⚠️ المتطلبات
* نظام تشغيل Windows 7 أو أحدث.
* Microsoft Excel (2010 أو أحدث).
* حزمة [VSTO Runtime](https://www.microsoft.com/en-us/download/details.aspx?id=48217) (إذا لم يظهر التبويب تلقائياً).
